# Awesome SUNXI

Awesome SUNXI - Gitee/Github/论坛上非常棒的全志平台开源项目，手册，数据集合。玩转全志方案必备！

## 项目结构
这里给出一个简单的项目结构图，需要查阅资料的可以在这些地方找到文档：

```
.
├─chips           # 芯片手册，封装等
│  ├─A133
│  ├─PMIC         # 电源管理芯片手册
│  ├─T113-s3
│  └─...
└─docs
    ├─linux-doc   # 通用linux软件手册 
    ├─melis-doc   # 通用melis软件手册 
    ├─tina-doc    # 通用tina软件手册 
    ├─v83x-doc    # 差异化v83x手册 
    └─...
```
